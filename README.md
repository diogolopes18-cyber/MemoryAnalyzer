# MemoryAnalyzer

A simple memory analyzer built with Rust

## Installation

1. To install Rust in your machine

    `$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

## Build

1. `cargo build`

2. `rustc src/main.rs`

## Usage

To use the memory analyzer CLI just run

`./main <filename>`
